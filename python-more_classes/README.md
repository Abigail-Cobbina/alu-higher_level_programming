# Python - More Classes

This project builds up a `Rectangle` class incrementally, task by task,
covering properties, string representation, instance counting, static
and class methods.

## Files

| File | Description |
|---|---|
| `0-rectangle.py` | Empty `Rectangle` class |
| `1-rectangle.py` | `Rectangle` with private width/height properties |
| `2-rectangle.py` | Adds `area()` and `perimeter()` |
| `3-rectangle.py` | Adds `__str__` for printable representation |
| `4-rectangle.py` | Adds `__repr__` (eval-able representation) |
| `5-rectangle.py` | Adds `__del__` (prints on instance deletion) |
| `6-rectangle.py` | Adds `number_of_instances` class attribute |
| `7-rectangle.py` | Adds `print_symbol` class attribute |
| `8-rectangle.py` | Adds `bigger_or_equal` static method |
| `9-rectangle.py` | Adds `square` class method |
